三、实验步骤与结果

  • 安装node-V7.7.1-x86.msi

    Git-2.12.0-32-bit.exe

  • 进入淘宝镜像

  • 复制使用说明

    npm install -g cnpm --registry=https://registry.npm.taobao.org

  • Windows cmd enter 拷贝到对话框

  • 继续输入命令：cd/d D:/MD

  • 继续输入命令：D： /MD>hexo init hexo,等待系统完成所有配置

  • 进入该目录在执行cnp install 等待安装完成

    ​ 命令1：/MD>cd hexo

    ​ 命令2：/MD>hexo cnpm istall

  • 输入 D:/mD>hexo hexo server

  • 输入http://locallhost:4000打开hexo网址
